# Lab PC Index

## 项目简介

`lab-pc-index` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的集合与索引，方便统一管理和访问。

## 目录结构

```
lab-pc-index/
├── index.html          # 主入口页面
├── pages/              # 存放各个独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源文件（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `index.html`：仓库的主页，提供页面索引和导航。
- `pages/`：存放所有独立 HTML 页面，每个页面为一个独立的网页文件。
- `assets/`：存放页面共用的样式、脚本、图片等资源文件。

## 页面归档说明

本仓库中的 HTML 页面主要用于实验、测试或临时归档。每个页面均为独立文件，不依赖外部服务或特定域名。页面内容可能包含：

- 前端功能演示
- 样式或布局测试
- 文档或笔记的静态展示

页面可能不定期更新或移除，请以仓库当前内容为准。

## 维护说明

- 本仓库由维护者不定期更新，无固定发布周期。
- 新增页面请放入 `pages/` 目录，并在 `index.html` 中添加对应链接。
- 公共资源请统一放置于 `assets/` 目录，避免重复引用。
- 不建议直接修改已归档页面的内容，如有需要可创建新版本页面。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `index.html` 即可看到页面列表。
3. 点击链接或直接访问 `pages/` 下的任意 HTML 文件。
